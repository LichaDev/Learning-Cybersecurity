# Introducción a la Seguridad Ofensiva

**Fecha:** 26 de Marzo, 2026
**Plataforma:** TryHackMe

## Conceptos Clave
En este primer módulo aprendí la diferencia fundamental entre la seguridad ofensiva y defensiva.

* **Offensive Security:** Se enfoca en "romper" o encontrar debilidades de forma ética para fortalecer el sistema antes de que un atacante real lo haga.
* **Ethical Hacking:** El uso de habilidades técnicas para identificar vulnerabilidades con permiso explícito.

## Laboratorio: FakeBank
En este ejercicio práctico, simulamos el acceso a un panel administrativo de un banco ficticio.

### Metodología seguida:
1.  **Exploración:** Navegación por el sitio web para entender su estructura.
2.  **Descubrimiento:** Identificación de rutas o parámetros que no deberían estar expuestos al público.
3.  **Acción:** Uso del simulador para ver cómo un atacante podría transferir fondos si el sistema no tiene controles adecuados.

### Lección Aprendida
La seguridad no se trata solo de tener una contraseña difícil, sino de no dejar "puertas traseras" o rutas administrativas expuestas que permitan saltarse la lógica del negocio.
